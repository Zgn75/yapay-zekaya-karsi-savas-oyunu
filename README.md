Öncelikle Oyunumuzu oynamak için bilgisayarınızda python yüklenmiş olmalıdır. "İndirilmesi gerekenler" klasöründen 3.9.0 bir sürüm indirebilirsiniz.

+Oyun nasıl açılır?

-Rar dosyasını indirdikten sonra klasörü masaüstüne atmalısınız. Ardından savas.py'ın olduğu klasöre girmelisiniz. Fakat bundan önce bazı modülleri yüklemeniz gerekir.


Gerekli modüller: time, random, keyboard,colorama


+Bunlar nasıl indirilir?

-Cevabı basit. "İndirilmesi gerekenler" klasöründen "download.bat"ı çalıştırarak modülleri tek tek indirebilirsiniz.


Modülleri indirdikten sonra savas.py'ın olduğu dosyaya geri dönüyoruz. savas.py dosyasına çift tıklayarak programı açabilirsiniz.


Herkese keyifli oyunlar dilerim. Not: Yazılım tecrübem daha 3-4 ay olduğu için en fazla bu kadar oluyor. Lütfen beni yargılamayın.
