Öncelikle Oyunumuzu oynamak için bilgisayarınızda python yüklenmiş olmalıdır.
Python.org sitesinden 3.x.x bir sürüm indirebilirsiniz.



Oyun nasıl açılır?
Rar dosyasını indirdikten sonra klasörü masaüstüne atmalısınız. Ardından savas.py'ın olduğu klasöre girmelisiniz.
Fakat bundan önce bazı modülleri yüklemeniz gerekir.



Gerekli modüller:
time
random
keyboard
Bunlar nasıl indirilir?


Cevabı basit. cmd'ye pip install "ModülAdı" yazarak modülleri tek tek indirebilirsiniz.


Modülleri indirdikten sonra savas.py'ın olduğu dosyaya geri dönüyoruz.
Dosyanın boş kalan herhangi bir kısmına mouse'ınızı getirip shift + sağ click yapıyoruz.

Tıkladıktan sonra orada ya "Powershell penceresini buradan açın." veya "Komut penceresini buradan açın." diyecektir. Sizde hangisi varsa ona tıklayın.
Ardından "python savas.py" yazdığınızda program çalışacaktır.
